# Тест 1. GitHub — основной инструмент управления ИТ-проектами

1. Что такое Pull Request?
   - [ ] Способ удалить ветку
   - [x] Предложение внести изменения в репозиторий
   - [ ] Автоматическая проверка кода

2. Что такое Commit?
   - [x] Сохранение версии проекта
   - [ ] Создание нового репозитория
   - [ ] Команда для слияния веток

3. Что означает Fork?
   - [x] Копия чужого репозитория в своём аккаунте
   - [ ] Удаление ветки
   - [ ] Слияние изменений

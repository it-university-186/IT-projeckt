# Тест 2. Канбан в GitHub

1. Что представляет собой колонка «In Progress»?
   - [ ] Задачи, которые уже завершены
   - [x] Задачи, над которыми ведётся работа
   - [ ] Архивные задачи

2. Что означает карточка в Kanban?
   - [x] Отдельная задача или issue
   - [ ] Релиз проекта
   - [ ] Коммит

3. Что происходит при перемещении карточки в колонку «Done»?
   - [x] Работа по задаче завершена
   - [ ] Создаётся новый проект
   - [ ] PR отклоняется

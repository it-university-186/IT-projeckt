# Итоговый экзамен — инструкция

Экзамен состоит из двух частей:
1. **Автотест** — несколько вопросов с выбором ответа (сверяются с `answers.json`).
2. **Ручная часть** — мини-эссе по кейсу (проверяет преподаватель в Issue).

## Как сдавать
1. В своём форке создайте ветку `final-exam/фамилия`.
2. Добавьте файл ответов: `students/<фамилия>/final_exam/answers.json` (пример ниже).
3. Добавьте мини-эссе: `students/<фамилия>/final_exam/essay.md`.
4. Откройте Pull Request в `main` этого репозитория.

### Пример `answers.json`
```json
{
  "1": "Fork → Ветка → PR → Автотесты → Ревью → Merge",
  "2": "Нужны правки перед принятием"
}
```

# Лабораторная №1: Знакомство с GitHub

## Цель
Освоить базовые операции GitHub и рабочий процесс курса.

## Задание
1. Сделайте **fork** репозитория дисциплины в свой аккаунт.  
2. Создайте ветку `lab01/<фамилия>` в своём форке.  
3. В каталоге `students/<фамилия>/lab01/` разместите:
   - `report.md` (краткий отчёт: что делали, скриншоты, ссылки на PR),
   - дополнительные файлы (по желанию).
4. Откройте **Pull Request** в этот репозиторий с ветки `lab01/<фамилия>`.
5. Дождитесь автопроверки и комментариев преподавателя.

## Критерии
См. `evaluation.md`.

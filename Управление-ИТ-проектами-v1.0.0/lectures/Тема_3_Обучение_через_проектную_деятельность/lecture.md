# Тема 3. Обучение через проектную деятельность

## 1. Концепция PBL (Project-Based Learning)
- Учимся через решение реальных задач.
- Роли в команде: лидер, аналитик, разработчик, тестировщик, дизайнер, документалист.

## 2. Постановка целей и результатов
- SMART-цели, критерии успеха, артефакты (MVP, отчёты, демо).

## 3. Структура проекта в GitHub
- Репозиторий проекта или папка `projects/<team>/`.
- Kanban, Issues, Wiki/Docs, Releases.

## 4. Коммуникации и код-ревью
- PR-шаблоны, чек-листы, Definition of Done.
- Митинги: планирование, стендапы, ретро.

## 5. Оценивание командной работы
- Индивидуальные вклады (коммиты/ревью).
- Командные артефакты (демо, релиз, документация).

# Лабораторная №2: Канбан-доска курса

## Задача
1. Создайте у себя GitHub Project (или проект в форке) со статусами: `To Do`, `In Progress`, `Submitted`, `Graded`.
2. Добавьте минимум 5 задач/Issue для своей учебной активности (lab, test, reading).
3. Настройте автоматизацию: перемещение карточек при открытии/закрытии PR.
4. Приложите скриншоты и короткий анализ метрик (lead/cycle time — оценочно).
